    迪拜阿赫利
    • 优 算上本赛季，迪拜阿赫利共7次参加亚冠赛事，在2015年还曾闯入亚冠决赛，不过不敌广州恒大屈居亚军。
    • 优 迪拜阿赫利阵内有两名实力非常强的巴西外援，分别是爱德华多和莱昂纳多，两人是球队的进攻源泉。
    • 优 迪拜阿赫利最近10个主场正式比赛有8场出大，大球率80%。
    • 劣 迪拜阿赫利已经接近半年没有踢过正式比赛，竞技状态存疑。
    沙尔霍德鲁
    • 优 伊朗球队沙尔霍德鲁本赛季是首次参加亚冠比赛，之前分别淘汰巴林球队艾尔里法和卡塔尔球队赛利亚跻身正赛。
    • 优 沙尔霍德鲁阵容比较寒酸，阵内只有一名乌克兰外援斯拉沃夫，这名球员身高接近两米，头球能力出色。
    • 劣 沙尔霍德鲁最近10个客场比赛仅赢了一场，胜率仅10%。
